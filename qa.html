Repository

The BioGRID Viewer's repository is at: https://github.com/svenmh/BioGRID-Viewer

Bugs should be reported to the Issues Page at GitHub. Modifications to this page should be pushed via git. Of course you can contact Sven with any issues or changes.

Queries

Generally queries should be done with the following format: http://biogrid-dev.princeton.edu/BioGRID-Viewer/ring.svg?searchNames=true&geneList=name&geneTaxIdList=taxid where name is the gene you are looking for and taxid is the species you are looking for. A list of species taxids supported can be received from the BGV/taxa.json file. But any REST options that returns a TAB2 files should do, report it if they don't.

Some common queries follows.

Nice Queries

  • Yeast CCC2
    &interSpeciesExcluded=TRUE
    Check that "Include Interspecies Interactions" is unchecked.
    &throughputTag=low
    Make sure the only "Throughput" checkbox that is checked is "Low".
  • Yeast PRB1
  • Human PRKACA
    Using REST's taxId
    should include disconnected interactions.
